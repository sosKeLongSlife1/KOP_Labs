# Simon Says

Навчальний проєкт гри **Simon Says**, реалізований на **React + Vite** в межах курсу з компонентно-орієнтованого програмування.

## Опис
Гра на запам’ятовування послідовностей. Гравець спостерігає за підсвіткою кольорових плиток та має відтворити послідовність у правильному порядку. З кожним рівнем складність зростає.

## Використані технології
- React
- Vite
- React Router
- Zustand
- CSS Modules
- Web Audio API (звук)

## Реалізовані етапи
- **Етап 1:** базова структура проєкту, компоненти та сторінки
- **Етап 2:** логіка гри з використанням кастомного хука
- **Етап 3:** налаштування гри, форма та модальне вікно (Portal)
- **Етап 4:** роутинг між сторінками з динамічним `userId`, стилізація
- **Етап 5:** керування станом через Zustand та таблиця результатів
